Quick description : This project involved designing a web application for the management of Office365 accounts at GEODIS. The main goals were to automate the account provisioning process, reduce errors, and ensure a seamless integration of data between HR and IT systems.

  • Research & Insights: Conducted interviews and workshops with key stakeholders (HR, IT, Managers) to identify pain points in the current process.
  • Solution Design: Created wireframes, prototypes, and user flows, using tools like Adobe XD and Material Design.
  • Impact: The application reduced data entry errors by 97%, streamlined processes, and improved data quality within the HR and IT systems.

The project was developed in phases, first rolling out in France, then expanding globally. I collaborated with the development team to ensure the final product met all user needs and business requirements.

Quick Description (Deferred Scheduler): This project focused on improving the deferred payment calendar for BoursoBank clients. A version already existed but lacked practicality, especially on mobile. The calendar was displayed in a table format, which was not responsive.

  • Research & Insights: We identified that the main issue for users, especially on mobile, was navigating through the large table of dates.
  • Solution Design: We switched from a table layout to a dropdown list, displaying the current month by default and allowing users to download the full-year calendar in PDF.
  • Impact: The redesign resulted in better user feedback, with a noticeable increase in the Play Store rating after the update.
